ON CONFLICT(content_hash) DO UPDATE SET copy_count = clipboard_entries.copy_count + 1, created_at = excluded.created_at, source_app = excluded.source_app, content_subtype = excluded.content_subtype, metadata = excluded.metadata, app_bundle_id = excluded.app_bundle_id, file_path = COALESCE(excluded.file_path, clipboard_entries.file_path)`。4. 在 `src-tauri/src/commands.rs` 的 `copy_to_clipboard` 里先调用诸如 `state.register_suppression_for_text(&content, 1500)` 的后端方法，再写系统剪贴板。`src/stores/clipboardStore.ts` 的 `copyToClipboard()` 必须改成 `invoke('copy_to_clipboard', { content })`，不再直接走 `writeText`。5. 在 `src-tauri/src/capture_runtime_tests.rs` 中除了解开 `test_capture_runtime_stop_cancels_tasks`、`test_capture_runtime_single_worker_and_suppression`、`test_capture_runtime_restart_is_single_owner` 外，还要新增 `test_capture_runtime_dedupe_migration_merges_existing_duplicates`，专门 seed 旧数据库重复行并验证迁移与唯一索引创建成功。</action>
